# SkillJava - Ejercicios de Estudio Java

Proyecto de práctica y aprendizaje de Java con ejercicios diarios.

## 📁 Estructura del Proyecto

```
SkillJava/
├── src/estudio/          # Código fuente Java
│   ├── ejercicio1.java   # Ejercicio 1: Entrada de datos por teclado
│   └── variables.java    # Ejercicio 2: Variables y tipos de datos
├── build/                # Archivos compilados
├── nbproject/            # Configuración de NetBeans
└── test/                 # Pruebas unitarias
```

## 🚀 Ejercicios Realizados

### Ejercicio 1: Entrada de Datos
- Uso de `Scanner` para entrada por teclado
- Variables: `byte` para documento y estrato, `String` para estado
- Entrada de documento, estado (A/S) y estrato (1-5)

### Ejercicio 2: Variables
- Declaración y asignación de variables `String`
- Uso de `System.out.println()` para salida

## 🛠️ Compilar y Ejecutar

```bash
# Compilar
javac -d build/classes src/estudio/*.java

# Ejecutar
java -cp build/classes estudio.ejercicio1
java -cp build/classes estudio.variables
```
